## How to install

```
wget https://github.com/rryter/node-status/releases/download/v0.0.1-alpha.6/lukso-status-v0.0.1-alpha.6-linux-amd64.tar.gz
tar -xf lukso-status-v0.0.1-alpha.6-linux-amd64.tar.gz
chmod +x lukso-status
./lukso-status
```
